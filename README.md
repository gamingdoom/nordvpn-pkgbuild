# nordvpn-pkgbuild
 A PKGBUILD and .install for nordvpn

# How Do I install?
In A Terminal, git clone this repo
```
$ git clone https://github.com/gamingdoom/nordvpn-pkgbuild.git
```
Then, cd into the resulting folder
```
$ cd nordvpn-pkgbuild
```
Finally, make & install the package
```
$ makepkg -si
```
